## Como executar na sua máquina e testar

<br>

### **Instalando o projeto**

No terminal digite todos os comandos abaixo:

```Bash
git clone https://github.com/devzgabriel/CEP-Inspector.git
```

```Bash
cd CEP-Inspector
```

```Bash
yarn install
```

```Bash
yarn start
```

### **Executando testes**

Os testes serão acrescentados em breve!
